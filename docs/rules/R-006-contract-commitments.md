# R-006 - Contract (Commitments)

- Phase: Contract selection

Contracts define additional obligations for the declarer.
They apply only to the Trump Game mode unless stated otherwise.

| Contract      | Description                                               |
|---------------|-----------------------------------------------------------|
| Game          | Declarer must score more than 50 points                   |
| Seven         | Declarer must win the last trick with the seven of trumps |
| Hundred       | Declarer must score at least 100 points                   |
| Hundred-Seven | Combination of Hundred and Seven                          |
